These resources will give you a solid grasp of the web development skills needed to use the GOV.UK Prototype Kit effectively.

Use them in addition to the tutorials and examples published on GOV.UK to get the most out of your prototypes.

Should I use Codecademy or W3Schools?

We feature resources from both of these platforms. You shouldn't have to do both and will probably quickly work out which you prefer:

  • Codecademy: easy to follow interactive courses. More resources are available at a cost.
  • W3Schools: easy to follow interactive guides, but in a different less modern style.

Where no free resources are available on Codecademy or W3Schools, we've identified alternative ones.

There are some other great platforms for tutorials if you are willing to pay: Treehouse for example.

Feel free to recommend other resources.

  2. 2. HTML

    You will need to learn HTML to use the GOV.UK Prototype Kit. It is the standard markup language for all web pages and is easy to learn.

    • Codecademy
    • W3Schools

  3. 3. CSS

    CSS is the language used to style an HTML document. It is useful to know a bit of CSS if you need to change the way components in the GOV.UK Design System look.

    • Codecademy
    • W3Schools

  4. 4. JavaScript

    JavaScript (JS) powers the dynamic features on most websites. In the Prototype Kit, it can be useful to know a little: for validating forms or changing the way a page behaves depending on a user's actions for example.

    • Codecademy - Introduction, Conditionals and Functions
    • W3Schools - up to 'JS Comparisons'

  5. 5. Other tools and skills

    • Git

      Git is the industry standard version control tool. Developers use it to store their code, collaborate and track all changes. You will need to know Git basics to access prototypes others have set up, save any changes you make to the code and deploy them.

      Learn Git - Codecademy

    • Terminal

      The Terminal application (sometimes referred to as 'the command line') is a text-only interface for your computer. Knowing a few basics will help you install, start and stop the Prototype Kit.

      Introduction to Terminal

    • Nunjucks

      Nunjucks is the templating framework used by the GOV.UK Prototype Kit. Nunjucks can provide shortcuts for common page elements so you don’t have to repeat the same chunks of HTML throughout a project - these are called 'macros'.

      Introduction to Nunjucks

    • Express

      is the web application framework used by the GOV.UK Prototype Kit. Knowing a little about will help you implement branching in your projects and understand the way the files in the kit are structured.

      Codecademy - Learn
